# 🏥 Medical Insurance Predictor AI

An end-to-end Machine Learning web application that predicts annual medical insurance charges based on user demographics and health habits.

## 🚀 Features
* **Accurate Predictions:** Uses Linear Regression trained on real-world medical data.
* **Interactive UI:** Built with Streamlit, allowing easy data input (Age, BMI, Children, Smoking habits).
* **Data-Driven:** Analyzes how life habits (like smoking) drastically change insurance costs.

## 🛠️ Tech Stack
* **Language:** Python
* **ML Libraries:** Pandas, Scikit-Learn
* **Web Framework:** Streamlit
